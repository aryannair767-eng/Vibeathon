## 📌 Project Overview
FitSync is a real-time fitness tracking web app built for two users — Aryan & Ved. It combines workout logging with competitive elements to make fitness more engaging. Users can log full workout sessions with advanced training metrics, track daily steps and calories, monitor bodyweight progress, and compete on a live leaderboard.

---

## ✨ Features
- 🏆 **Live Leaderboard** — Aryan vs Ved with points, streaks, and a battle bar
- 💪 **Workout Session Logger** — log exercises with sets, reps, weight, RPE & RIR
- 🔥 **Smart Calorie Estimation** — factors in volume, exercise type, RPE, RIR & bodyweight
- 👟 **Daily Steps & Calorie Tracker** — with progress bars and daily goals
- ⚖️ **Weight Tracking** — log bodyweight, set a target, and view a progress chart
- 🌙 **Dark & Light Mode** — toggle between dark (black + blue) and light (white + blue)
- ↩️ **Undo & Remove** — remove any exercise or undo the last removal
