# FitSync ⚡
A competitive fitness tracker built for Aryan & Ved.
Built with Python + Streamlit for Vibeathon hackathon.

## Features
- 💪 Workout session logger with RPE & RIR tracking
- 🏆 Aryan vs Ved live leaderboard
- 🔥 Calorie estimation based on volume, RPE & RIR
- ⚖️ Weight tracking with progress chart
- 👟 Daily steps & calorie goal tracker
- 🌙 Dark & light mode toggle

## Run locally
pip install streamlit
streamlit run fitsync_app.py
